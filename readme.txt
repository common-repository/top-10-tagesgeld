Contributors: finango
Requires at least: 2.1
Tested up to: 2.8.6

TOP 10 Tagesgeld stellt ein Widget zur Verfügung, um die 10 besten Tagesgeld-Angebote und Zinssätze anzuzeigen.

== Description ==

Funktionalität:

- Macht ein Widget verfügbar, um in der Sidebar die 10 besten Tagesgeld-Angebote und deren Zinssätze anzuzeigen.
- Die Daten werden vom Server http://www.toptentagesgeld.de gelesen.

Changelog:

* 1.5.1 Bugfix: Auf manchen Servern wurde das Aktivieren des Plugins mit einem "...unexpected $end..."-Fehler quittiert.
* 1.5	Erste veröffentlichte Version

== Installation ==

1. Laden Sie das Plugin-Archiv herunter und entpacken Sie es.
2. Kopieren Sie die Datei top10tagesgeld.php nach wp-content/plugins/.
3. Öffnen Sie in der Wordpress-Administration den Bereich Plugins und aktivieren Sie das Plugin "TOP 10 Tagesgeld".
4. Öffnen Sie in der Wordpress-Administration den Bereich Design / Widgets und klicken Sie in der Spalte "Verfügbare Widgets" neben dem Eintrag "TOP 10 Tagesgeld" auf "Hinzu".
5. Passen Sie bei Bedarf den Sidebar-Titel des Widgets an, indem Sie nach dem Hinzufügen des Widgets zur Sidebar in der rechten Spalte der Widget-Administration neben dem Widget "TOP 10 Tagesgeld" auf "Bearbeiten" klicken, einen Titel im Feld "Titel" eingeben und "Fertig" klicken.
